Scrabble

Scrabble is a classic word game that challenges players to create words on a game board using letter tiles. Players earn points based on the length and complexity of the words they create. Scrabble is a game of strategy, vocabulary, and skill, and is enjoyed by players of all ages.

How to play

Scrabble is played by placing letter tiles on a game board to create interconnected words. Each player starts with a set of letter tiles and takes turns placing tiles on the board to create words. Players can earn extra points by using tiles with higher point values and placing tiles on special squares that offer bonus points. The game continues until all letter tiles have been used or no more valid words can be created. The player with the highest score at the end of the game wins!
